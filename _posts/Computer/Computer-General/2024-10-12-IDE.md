### 🫧 Visual Studio Code - Shortcut | 단축키

- Intellisense \| 인텔리센스 : `Ctrl + Space`
- Auto Format \| 자동완성 : `Shift + Alt + F`
- 줄 들여쓰기 : `Ctrl + [`, `Ctrl + ]`
- Tab Moves Foucus : `Ctrl + M`
  - 탭 키 의도대로 되지 않을 때 확인할 것

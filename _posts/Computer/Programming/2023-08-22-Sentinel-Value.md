## 💫 Sentinel Value

---

(Flag Value, Trip Value, Rogue Value, Signal Value, Dummy Data)  

루프, 재귀, 데이터의 종료를 나타내기 위해 사용하는 특수한 값  

- 예시
  - C 문자열의 끝을 나타내는 Null (Char)
  - 연결 리스트의 끝을 나타내는 Null Pointer

## 💫 [Github Desktop Commit Amend, Undo, Revert](https://bloodstrawberry.tistory.com/832)

---

- `Amend`: Commit 이름, 설명 수정
- `Undo`: Commit 취소 (변경 내용은 바뀌지 않음)
- `Revert`: Commit 변경 내용을 되돌리는 또 다른 Commit 생성

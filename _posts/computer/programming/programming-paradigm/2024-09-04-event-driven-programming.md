## Event Driven Programming | 이벤트 기반 프로그래밍

---

컴퓨터에서 발생하는 이벤트에 반응하도록 프로그램을 만드는 것.  

GUI (Graphical User Interface) 프로그래밍에 특히 유용하다.  

## Event | 이벤트 (사건)

---

사건 ?  

- 사용자의 입력 (클릭, 키보드 입력 등)
- 이메일 도착
- 파일 복사 \| 다운로드 완료
- 새로운 외부 장치 연결

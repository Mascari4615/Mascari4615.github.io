---

### **🫧 Example Input**

```cpp
12345669999 // 방 번호
```

### **🫧 Example Output**

```cpp
3
// 0 ~ 9 카드 세트 최소 몇 개를 가지고 방 번호를 만들 수 있는가
// 이때, 6은 뒤집어서 9로, 9는 뒤집어서 6으로 쓸 수 있음
```

<br>

<!-- ---- ---- ---- ----  ---- ---- ---- ----  ---- ---- ---- ----  ---- ---- ---- ---- -->

## **💫 C++ 풀이**

---

```cpp
#include <iostream>
#include <algorithm>

using namespace std;

int main()
{
	ios::sync_with_stdio(false);
	cin.tie(nullptr);

	string n;
	cin >> n;

	int count[10];
	fill(count, count + 10, 0);

	for (auto c : n)
		count[c - '0']++;

	int max = -1;
	for (int i = 0; i < 10; i++)
	{
		// MISS
		// 실수로 빼먹음
		if ((i == 6) || (i == 9))
			continue;
		
		if (max < count[i])
			max = count[i];
	}

	if (max < (count[6] + count[9] + 1) / 2)
		max = (count[6] + count[9] + 1) / 2;

	cout << max;
}
```

## **💫 C++ 풀이**

---

```cpp
#include <iostream>
#include <algorithm>

using namespace std;

int main()
{
	ios::sync_with_stdio(false);
	cin.tie(nullptr);

	string s;
	cin >> s;

	int count[26];
	fill(count, count + 26, 0);

	for (char c : s)
		count[c - 'a']++;

	for (int i = 0; i < 26; i++)
		cout << count[i] << ' ';
}
```

## **💫 [문제](https://www.acmicpc.net/problem/10807)**

---

### **🫧 Example Input**

```cpp
11 // n : 정수의 개수 (1 <= n <= 100)
1 4 1 2 4 2 4 2 3 4 4 // 정수들 (-100 <= 정수 <= 100)
2 // v : 찾으려고 하는 정수 (-100 <= v <= 100)
```

### **🫧 Example Output**


## **💫 C++ 풀이**

---

```cpp
#include <iostream>
#include <algorithm>

using namespace std;

int main()
{
	ios::sync_with_stdio(false);
	cin.tie(nullptr);

	int n, a, v;
	cin >> n;

	// -100 ~ 100 범위를 0 ~ 200 으로
	int freq[205];
	fill(freq, freq + 205, 0);

	for (int i = 0; i < n; i++)
	{
		cin >> a;
		freq[a + 100]++;
	}

	cin >> v;

	cout << freq[v + 100];
}
```
